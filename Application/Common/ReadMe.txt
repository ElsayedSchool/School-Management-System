1- interfaces is used to add interfaces for each externals exist in infrastructure layer

2- Models is used to customize response for specific query or return only specific data to the user

3- Validators are used to validate input Dtos before using it inside our logic

4- Mapping is used to map <from Dto to entity or viseversa>

5- Exceptions is used to create Custom Validation for possible Command failure 
	Sucha as(not Found, not created) and so on
